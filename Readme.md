## Useful commads for virtual environment
https://fastapi.tiangolo.com/virtual-environments/#create-a-virtual-environment

### Create .venv
```
$ python -m venv -venv
```
### Initialize .venv
```
$ source .venv/bin/activate
```
### Verify .venv
```
$ which python
```
> Output: /home/user/code/awesome-project/.venv/bin/python
